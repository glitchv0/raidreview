# Raid Reports

Performance analysis reports for **Make Azeroth Great Again** guild raids (WoW Classic - Mists of Pandaria).

## View Reports

- **GitHub Pages:** https://glitchv0.github.io/raidreview/
- **Custom Domain:** https://glitchv0.com/raidreview/

## About

Reports are generated from [Warcraft Logs](https://classic.warcraftlogs.com/) data and include:

- Per-player, per-boss performance analysis
- Ability rotation comparison vs top performers
- Talent and glyph recommendations
- Consumable audit
- Auto-generated improvement suggestions
